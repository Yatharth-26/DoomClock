# DoomClock
⏳ A procrastination killer app that uses your life clock and existential dread to make you actually get things done.
## Objectives of the Project
The objectives of this project are:
- To build a web application using HTML, CSS, and JavaScript
- To integrate public APIs using fetch
- To implement search, filter, and sort using Array Higher Order Functions
- To store user data using local storage
- To create a responsive user interface
- To deploy the project online

---

## APIs Used
The following public APIs are used in this project:

1. Quotable API  
   Used to fetch motivational quotes.  
   https://api.quotable.io/random

2. Bored API  
   Used to fetch random activity suggestions.  
   https://www.boredapi.com/api/activity

3. Advice Slip API  
   Used to fetch advice messages.  
   https://api.adviceslip.com/advice

---

## Features of the Application
The application will include the following features:

### Core Features
- Display life clock based on user's birthdate
- Add tasks with deadlines and priority
- Mark tasks as completed
- Track overdue tasks
- Display time lost due to missed deadlines
- Show motivational quotes
- Suggest random activities
- Show advice messages

### Interactive Features
- Search tasks by keyword
- Filter tasks by status (Completed, Pending, Overdue)
- Sort tasks by deadline or priority
- Dark mode / Light mode toggle

### Additional Features
- Local storage to save tasks and user data
- Loading indicators during API calls
- Responsive design for mobile and desktop

---

## Array Higher Order Functions Used
This project uses Array Higher Order Functions for data manipulation:

- map() → Used to display tasks
- filter() → Used for searching and filtering tasks
- sort() → Used to sort tasks by deadline or priority
- find() → Used to find specific tasks
- reduce() → Used to calculate total time lost

Traditional loops like for and while will not be used for search, filter, and sort operations.
